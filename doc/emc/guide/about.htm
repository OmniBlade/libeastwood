What's this?

Dune 2 (TM) comes along with EMC files that are binary files of scripts written in internal language.
Programs included in the package allow to alter the content of the scripts in easy way just by using any text editor (e.g. Notepad).
Note: it is advised to have some programming knowledge because the guide doesn't explain all terms it uses.
