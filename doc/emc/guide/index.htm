EMC Scripts Blitz Guide

This guide describes how to create custom EMC scripts to use in Dune 2. It is advised to really feel like learning EMC :-)

EMC Script is language used to organize tasks that should be executed during the lifetime of specified object type (object stands for building, unit or team) that appears in Dune 2.

This guide is designed as a couple of lessons which rise you to successive tiers of experience (like in Quake 3 hehe :))

  1. tier 1:
    • Stack and Math operations
  2. tier 2:
    • External procedures and Return code
    • Jumps, loops and Condition codes
  3. tier 3:
    • Custom procedures, Arguments and Variables
  4. appendice:
    • Global registers
    • External procedures reference
    • Cheats
